# Источники данных

Внешние системы и наборы данных, используемые ALTEKO. Включает тип доступа, содержание данных и статус интеграции.

---

## Публичные реестры

### Kadastrs.lv
**Провайдер:** Valsts zemes dienests (Государственная земельная служба)
**Доступ:** Открытые данные (data.gov.lv, CC BY) + контрактный API (требует соглашения с VZD)
**Данные:**
- Адрес и кадастровый номер здания
- Год постройки
- Общая площадь
- Серия здания (для старых домов)
- Класс энергоэффективности (A–G) — через отдельный набор BVKB
- Количество этажей, квартир
**Использование в ALTEKO:** Сегментация зданий для бенчмаркинга; входные данные для ИИ-прогноза
**Примечания:** Открытые данные Building.ZIP (еженедельное обновление) — основной путь без контракта. Контрактный API VZD — для более богатых данных в реальном времени, оценить после MVP.

---

### Zemesgrāmata
**Провайдер:** Tiesu administrācija (Судебная администрация)
**Доступ:** Только веб-портал; ~€5 за запрос; пакетного API нет
**Данные:**
- Актуальные собственники квартир (имя, доля %)
- История сделок
- Обременения (ипотеки, ограничения)
**Использование в ALTEKO:** Верификация права на голосование; взвешивание долей собственности в протоколах голосования
**Примечания:** Программного пакетного API не существует. Для Фазы 3 (голосование): ручная загрузка CSV правлением товарищества (предпочтительно) или нотариальная выписка.

---

### Altum.lv
**Провайдер:** ALTUM (государственный институт развитого финансирования)
**Доступ:** Публичный сайт; подтверждённого структурированного API нет
**Данные:**
- Действующие субсидийные программы и условия
- Допустимые типы зданий
- Требования к заявке и контрольные списки документов
- Список одобренных проектов (публичный)
**Использование в ALTEKO:** Расчёт права на субсидию; генерация конвейера документов
**Примечания:** Данные, вероятно, потребуют ручного обновления или скрейпинга. Открытого API не подтверждено.

---

### CSP.gov.lv
**Провайдер:** Centrālā statistikas pārvalde
**Доступ:** Публичные наборы данных (CSV/XLSX для скачивания)
**Данные:**
- Индексы строительных затрат (по году, региону)
- Статистика жилищного фонда
- Демографические данные
**Использование в ALTEKO:** Скорректированные на инфляцию оценки стоимости ремонта; оценка объёма рынка
**Примечания:** Подходит для пакетного скачивания, не для интеграции в реальном времени.

---

### GeoRiga / GeoLatvija
**Провайдер:** Рижская городская дума / VSIA «Latvijas Ģeotelpiskās informācijas aģentūra»
**Доступ:** opendata.riga.lv / geolatvija.lv (частично открытый)
**Данные:**
- Контуры зданий и геопространственные координаты
- Границы районов
- Реестр адресов
**Использование в ALTEKO:** Сегментация по районам; визуализация на карте
**Примечания:** Статус доступа к конкретным наборам данных требует проверки.

## Платные провайдеры данных

### Lursoft
**Провайдер:** Lursoft IT (коммерческий)
**Доступ:** Платная подписка / API
**Данные:**
- Данные о регистрации компании
- Годовая финансовая отчётность
- Структура собственности юридических лиц
- Показатели платёжного поведения
**Использование в ALTEKO:** Верификация подрядчиков — проверка финансового состояния перед включением в маркетплейс
**Примечания:** Платный API. Стоимость и условия требуют коммерческого соглашения.

---

### Jāņa sēta API
**Провайдер:** Jāņa sēta (коммерческий, kartes.lv)
**Доступ:** API-ключ (бесплатный тариф доступен, email: kartes@kartes.lv)
**Данные:**
- Автодополнение адреса (поиск по строке)
- Геокодинг (адрес → lat/lon + vzd_id)
- Обратное геокодирование с кадастровым поиском
- API квартир (номера квартир по vzd_id здания)
**Использование в ALTEKO:** Основной провайдер для автодополнения адреса и геокодинга в модуле поиска по адресу
**Примечания:** Оборачивает официальные данные VAR (реестр адресов VZD) через чистый REST API.

## Пользовательские данные

### PDF с расходами (загружаются пользователями)
**Источник:** Ежемесячные/квартальные отчёты управляющих компаний
**Формат:** PDF (сканированный или цифровой)
**Обработка:** OCR → LLM-парсинг → структурированная таблица расходов
**Извлекаемые данные:**
- Отопление (siltumenerģija) — €/мес., €/м²
- Холодная вода, горячая вода — м³, €
- Водоотведение
- Уборка / обслуживание территории
- Взносы в ремонтный фонд здания
- Административные сборы
**Примечания:** Основной канал сбора данных для формирования базы бенчмаркинга. Качество сильно варьируется между управляющими компаниями.

## Погода и климат

### Weather API
**Рекомендуемый провайдер:** Open-Meteo (бесплатно, open-source)
**Необходимые данные:**
- Исторические градусо-дни отопления (HDD) по местоположению
- Среднемесячные температуры по районам
**Использование в ALTEKO:** ИИ-прогноз тепловой экономии (моделирование теплопотребления)
**Примечания:** Open-Meteo предоставляет исторические данные бесплатно с достаточным покрытием для Латвии.

| Altum.lv | Высокий | Запланировать ручное обновление или скрейпинг |

| CSP.gov.lv | Средний | Пакетное скачивание, периодическое обновление |
