# Ключевые факты и цифры

Количественные утверждения, используемые в документации по планированию ALTEKO.
Каждый факт помечен: **[ВЕРИФИЦИРОВАНО]** (с источником) или **[НЕ ВЕРИФИЦИРОВАНО]** (присутствует в заметках, источник неизвестен).

Не верифицированные факты нельзя использовать в материалах для инвесторов или публичных коммуникациях без источника.

## Оставшиеся неверифицированные данные

Пункты, всё ещё не имеющие первичного источника:

1. Расхождение показаний счётчиков 10–25% → проверить в отчётах Latvijas Ūdens или SPRK
2. Потери тепла в 3 раза выше, чем в новых домах → проверить в данных энергетического аудита Министерства экономики
3. Отопление = до 60% зимнего бюджета → проверить в обследованиях домохозяйств CSP
4. Средняя стоимость реновации €200–500 тыс. → верифицировать по данным закрытых проектов ALTUM
5. 12+ шагов для заявки Altum → посчитать по актуальному руководству на altum.lv
