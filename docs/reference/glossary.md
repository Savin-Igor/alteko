# Глоссарий

Латвийские термины, организации и технические понятия, используемые в документации ALTEKO.

---

## Организации и программы

**Altum**
Государственный институт развитого финансирования Латвии. Предоставляет субсидированные кредиты и гранты для энергоэффективной реновации многоквартирных домов. Требует согласия ≥50% собственников и покрывает 40–50% допустимых расходов на ремонт.
Сайт: altum.lv

**CSP** (Centrālā statistikas pārvalde)
Центральное статистическое управление Латвии. Публикует индексы строительных затрат, жилищную статистику и демографические данные.
Сайт: csp.gov.lv

**Kadastrs** (Valsts zemes dienests — kadastra informācijas sistēma)
Государственный кадастр недвижимости Латвии. Содержит параметры зданий: адрес, год постройки, общую площадь, класс энергоэффективности (A–G), серию здания. Публичный доступ через kadastrs.lv.

**Lursoft**
Коммерческий провайдер данных о компаниях. Содержит финансовую отчётность, регистрационные данные и сведения о собственниках латвийских юридических лиц. Используется для верификации финансового состояния подрядчиков. Платный API.
Сайт: lursoft.lv

**Zemesgrāmata** (Valsts vienotā datorizētā zemesgrāmata)
Земельный реестр (права собственности) Латвии. Содержит актуальные списки собственников по каждой квартире, доли собственности и обременения. Используется для верификации права голоса и структуры собственности. Публичный доступ.
Сайт: zemesgramata.lv

## Жилищная сфера и недвижимость

**Biedrība** (dzīvokļu īpašnieku biedrība)
Жилищное товарищество — латвийский аналог ТСЖ (товарищества собственников жилья). Юридическое лицо, созданное собственниками квартир для коллективного управления домом. Может заключать договоры, открывать банковские счета и подавать заявки на субсидии независимо от управляющей компании.

**Dzīvojamā māja / Daudzdzīvokļu māja**
Жилой многоквартирный дом. Основная единица анализа в ALTEKO.

**Серии зданий**
Советские сборные панельные серии. Каждая серия имеет определённые параметры конструкции (толщина стен, планировки, тип утепления):

| Серия | Годы постройки | Этажность | Примечания |
|-------|----------------|-----------|------------|
| 103   | 1960-е–70-е    | 5         | Хрущёвки |
| 104   | 1960-е–70-е    | 5         | Вариант серии 103 |
| 119   | 1970-е–80-е    | 9         | Самая распространённая в пригородах Риги |
| 467   | 1970-е–80-е    | 5–9       | Серия литовского происхождения |
| 602   | 1980-е         | 5–9       | Поздняя брежневская эпоха |

**Класс энергоэффективности**
Латвийская система сертификации энергоэффективности зданий:

| Класс | Потребление (кВт·ч/м²/год) | Значение |
|-------|---------------------------|---------|
| A     | < 95                      | Почти нулевое энергопотребление |
| B     | 95–150                    | Очень эффективный |
| C     | 150–200                   | Эффективный |
| D     | 200–250                   | Стандартная новостройка |
| E     | 250–300                   | Ниже стандарта |
| F     | 300–400                   | Плохой |
| G     | > 400                     | Очень плохой — типичный советский дом |

**Доля собственности**
Право собственности измеряется долей от общей площади здания. Вес голоса пропорционален доле собственности, а не количеству квартир. Верифицируется через Zemesgrāmata.

## Управление и обслуживание

**Apsaimniekotājs** (управляющая компания)
Компания по управлению недвижимостью, отвечающая за обслуживание здания, выставление счетов за коммунальные услуги и координацию подрядчиков. Крупные операторы в Риге: RNP (Rīgas namu pārvaldnieks), Latvijas Namsaimnieks, Latvijas namu pārvalde.

**RNP** (Rīgas namu pārvaldnieks)
Крупнейшая управляющая компания в Риге, управляющая ~7 000 домами. Муниципальное предприятие. Основной источник непрозрачных PDF-счетов, на которые нацелен модуль аудита ALTEKO.

**Apsaimniekošanas maksa**
Ежемесячная плата за управление (€/м²), взимаемая управляющей компанией. Сильно варьируется между домами — основной объект выявления аномалий при бенчмаркинге.

## Цифровая идентификация

**Smart-ID**
Мобильное приложение для электронной идентификации и цифровой подписи, широко используемое в Латвии и Эстонии. Обеспечивает юридически обязывающую дистанционную аутентификацию. Необходим для юридически значимого электронного голосования в ALTEKO.

**eParaksts** (электронная подпись)
Латвийская национальная система электронной подписи. Две формы:
- eParaksts mobile — на основе приложения, аналог Smart-ID
- eParaksts USB-токен — физическое аппаратное устройство

Оба юридически признаны для подписания договоров и официальных документов.

## Финансовые термины

**Siltumenerģija tarifs**
Тариф на централизованное теплоснабжение. Устанавливается поставщиком тепла (Latvijas Gāze, Rīgas Siltums и др.) и утверждается регулятором (SPRK). Публикуется в открытом доступе.

**SPRK** (Sabiedrisko pakalpojumu regulēšanas komisija)
Комиссия по регулированию общественных услуг. Устанавливает предельные тарифы на тепло, воду и электроэнергию.

## Технические термины

**GeoRiga / GeoLatvija**
Рижский городской портал открытых данных. Содержит геопространственные наборы данных: контуры зданий, адреса, муниципальная инфраструктура.
Сайты: geolatvija.lv / opendata.riga.lv

**OCR** (Optical Character Recognition, оптическое распознавание символов)
Технология извлечения текста из сканированных PDF и изображений. В ALTEKO: применяется к счетам и отчётам о расходах управляющих компаний для преобразования в структурированные данные.

**LLM** (Large Language Model, большая языковая модель)
Модель ИИ для семантического парсинга текста, извлечённого OCR: категоризации статей расходов, нормализации форматов, выявления аномалий выставления счетов.

**Движок бенчмаркинга**
Основной аналитический компонент ALTEKO. Рассчитывает медианные значения расходов по сегменту здания (серия × район × диапазон площади) и помечает отклонения выше заданных порогов (P75 или >15%).

**P25 / P75 (перцентиль)**
Статистические маркеры: P25 — нижний квартиль (25% домов тратят меньше), P75 — верхний квартиль (25% домов тратят больше). Используются для определения «нормального диапазона» при бенчмаркинге расходов.
