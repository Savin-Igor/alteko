# Бизнес-концепция

## Проблема

В Латвии ~23 500 советских многоквартирных домов, нуждающихся в реновации. Более половины жилищного фонда — панельные дома серий 103, 104, 119, 467, 602, построенные в 1960–1980-е годы. У них нет нормального утепления, устаревшие системы отопления и класс энергоэффективности F или G.

Это порождает две связанных проблемы.

### Проблема 1: Непрозрачные счета

Жители получают ежемесячные PDF-квитанции от управляющих компаний (apsaimniekotāji), но не могут проверить, справедлива ли цена. Одна и та же статья расходов может отличаться в 2–3 раза между соседними домами идентичной конструкции. Инструментов для сравнения нет.

Конкретные проблемы:
- Расхождение показаний счётчиков: общедомовой счётчик стабильно показывает на 10–25% больше, чем сумма квартирных. Разница выставляется жителям без объяснений.
- Непрозрачные статьи обслуживания: бумажные акты, непроверяемые фотографии, произвольные взносы в «фонд ремонта».
- Нет бенчмаркинга: жители не могут сравнить свой дом с домами той же серии, года постройки и района.

### Проблема 2: Барьер для реновации

Государство (через Altum) субсидирует 40–50% стоимости энергоэффективной реновации. Но конверсия составляет лишь 2,7% — 624 завершённых здания из 23 500 подходящих. Причины:

1. **Сложность процесса:** 12+ бюрократических шагов для получения финансирования Altum
2. **Требование согласия:** нужно ≥50% голосов собственников — в доме из 80 квартир с отсутствующими владельцами это почти невозможно без инструментов
3. **Информационный разрыв:** жители не знают, сколько реально будет стоить реновация в месяц, что покрывает субсидия и какова будет экономия
4. **Непрозрачность подрядчиков:** нет надёжного способа найти, оценить и сравнить компании по ремонту конкретного типа домов

---

## Решение: ALTEKO

ALTEKO — двухмодульная платформа, которая превращает раздражение жителей (непрозрачность счетов) в действие (капитальный ремонт).

**Модуль 1 — Аудит (воронка привлечения):**
Жители загружают PDF-отчёт о расходах. ALTEKO парсит его через GPT-4o vision единственным вызовом (без отдельного OCR-шага), сравнивает с бенчмарками для идентичных домов (по серии, району, площади) и показывает, где именно дом переплачивает и насколько.

**Модуль 2 — Реновация (бизнес-движок):**
Когда аудит выявляет завышенные расходы на отопление — а это будет почти для каждого дома класса F/G — ALTEKO предлагает путь: рассчитать точную экономию после реновации, показать сумму субсидии Altum, сгенерировать все необходимые документы, собрать юридически значимые голоса собственников и связать дом с проверенными подрядчиками.

## Цепочка ценности

```
ДАННЫЕ → ИНСАЙТ → НАМЕРЕНИЕ → ДЕЙСТВИЕ → СДЕЛКА → ВЫРУЧКА
```

1. **ДАННЫЕ:** Жители загружают счета → ALTEKO строит базу расходов по тысячам домов
2. **ИНСАЙТ:** Бенчмаркинг показывает: «ваш дом платит за тепло на 40% больше похожих домов»
3. **НАМЕРЕНИЕ:** ИИ-прогноз: «после реновации счёт за отопление упадёт с €250 до €110 в месяц»
4. **ДЕЙСТВИЕ:** Платформа собирает голоса собственников и генерирует юридически значимые протоколы
5. **СДЕЛКА:** Подрядчик выигрывает контракт на ремонт €200–500 тыс. через маркетплейс
6. **ВЫРУЧКА:** ALTEKO зарабатывает комиссию 1.5% = €3–12 тыс. с дома

## Почему сейчас

Три сходящихся фактора делают этот момент подходящим:

1. **Зелёный мандат ЕС:** Растут субсидии на утепление зданий по всему ЕС. Программа Altum расширяется, а не сворачивается.
2. **Давление МВФ:** Латвия под давлением в части цифровизации государственных услуг и строительных процессов. Есть официальная поддержка платформ, снижающих бюрократические барьеры.
3. **Инфраструктура цифровой идентификации:** Smart-ID и eParaksts уже широко распространены в Латвии. Юридически значимое электронное голосование технически возможно сегодня — 5 лет назад его не было.
4. **Нет конкурента:** Ни одна латвийская платформа не объединяет аудит расходов + воронку реновации + электронное голосование в одном продукте.

## Ключевой инсайт

> Пользователи приходят за ответом на вопрос «почему у меня такой высокий счёт» — не за услугами реновации.
> Реновация — это то, что происходит после того, как доверие выстроено через прозрачность.

Модуль аудита — не убыточный лидер продаж, это канал привлечения. Каждый дом, загрузивший счёт, становится лидом для реновации, как только аномалия выражена в цифрах.

## Конкурентный ров

Устойчивое конкурентное преимущество — база расходов: исторические данные о выставленных счетах по тысячам латвийских домов, сегментированные по серии, району, площади и периоду. Эта база:

- Становится точнее по мере загрузки данных новыми домами
- Не может быть скопирована управляющими компаниями (у них данные только по своему портфелю)
- Не может быть скопирована государством (у него есть метаданные о зданиях, но не детали счетов)
- Обеспечивает бенчмаркинг, невозможный без этой конкретной комбинации данных

Этот ров формируется за 2–3 года с момента выхода на рынок.
