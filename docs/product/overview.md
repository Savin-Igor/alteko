# Обзор продукта

## Что делает ALTEKO

ALTEKO — платформа для жителей и товариществ советских многоквартирных домов в Латвии. Она объединяет две функции:

1. **Аудит расходов** — парсинг счетов управляющих компаний, сравнение с бенчмарками, выявление переплат и аномалий
2. **Сопровождение реновации** — расчёт экономии, навигация по субсидиям Altum, сбор голосов собственников, подбор подрядчиков

Два модуля — не отдельные продукты. Аудит — воронка привлечения для реновации.

## Путь пользователя

Путь пользователя строится из трёх эмоциональных этапов.

### Этап 1: Злость («Почему так дорого?»)

Пользователь загружает ежемесячный PDF-счёт от управляющей компании.

**Шаг 1 — Превью (без регистрации):**

ALTEKO парсит счёт и сразу показывает одно число:
> «Ваш дом серии 119 в Пурвциемсе платит за отопление на 40% больше, чем медиана похожих домов в том же районе.»

Это зацепка. Пользователь пришёл за подтверждением подозрения — и получает его мгновенно.

**Шаг 2 — Email-гейт:**

Под ключевой цифрой появляется форма:
> «Укажите email — получите полный отчёт с разбивкой по всем категориям, сравнением с похожими домами и списком аномалий.»

Email вводится один раз. После этого все последующие отчёты приходят автоматически.

**Шаг 3 — Полный отчёт (на email + в личном кабинете):**

- Разбивка по всем категориям (отопление, вода, уборка, ремонт, администрирование)
- Сравнение с P25/P50/P75 по той же серии + район
- Все выявленные аномалии: «Потребление горячей воды на 23% выше медианы — возможная утечка в доме»

### Этап 2: Надежда («Сколько я мог бы сэкономить?»)

Пользователь нажимает: «Что изменит реновация?»

Email уже введён на Этапе 1 — гейта больше нет. Пользователь, зашедший по ссылке из письма, уже идентифицирован.

ALTEKO показывает:
> «Исходя из класса энергоэффективности вашего дома (F), площади (4 200 м²) и текущих расходов на отопление: после реновации счёт за отопление снизится с €250 до €110 в месяц.»
> «Субсидия Altum покрывает 45% от общей стоимости. Ваша доля: €3 400. Срок окупаемости: 4,2 года.»

Дополнительно:
- Ежемесячная экономия (€/мес. для средней квартиры)
- График ROI
- Оценка влияния на стоимость недвижимости

### Этап 3: Действие («Начинаем»)

Пользователь нажимает: «Начать процесс реновации»

ALTEKO предоставляет:
- Кампанию электронного голосования для сбора ≥50% согласия собственников через Smart-ID/eParaksts
- Автоматически сгенерированный протокол собрания (соответствует требованиям Altum)
- Заявку на субсидию, предзаполненную данными дома
- Список верифицированных подрядчиков с рейтингами и оценками для этого типа домов

## Язык

Продукт должен быть полностью двуязычным: **латышский и русский**.

В советских домах высокая доля русскоязычных жителей, особенно пенсионеров. Достичь ≥50% согласия собственников для голосования невозможно без поддержки русского языка. Интерфейс на одном языке не позволит набрать нужный порог ни в одном доме.
