# Модуль: Finansējuma scenāriji / Финансовые сценарии

> Старое название: «Модуль реновации». В коде остаётся `src/app/[locale]/renovation/`, но в UI переименовано согласно `readiness-glossary.md`.

## Назначение

Показать дому **5 реалистичных сценариев финансирования** ремонта/реновации с честной оценкой пригодности и пометкой статуса окна.

В v1 модуль обещал «получите субсидию ALTUM» с одним калькулятором. В v2 модуль показывает карту вариантов с явным указанием, какое окно открыто, какое закрыто, какое ожидается.

**Главное правило:** ALTEKO не обещает финансирование. ALTEKO показывает **provizoriskais novērtējums** / **предварительную оценку**.

## Пять сценариев

### 1. Sociālā klimata fonds 2026-2032 (SCF)

| Параметр | Значение |
|----------|----------|
| Тип | Грант / капитальная скидка |
| Объём для МКД | €127M публичных + €31.75M нац. = €186M общих |
| Окно | `Gaidāms` / Ожидается (MK noteikumi Q4 2026, заявки до Q2 2027) |
| Целевая категория | F-класс (≥125 kWh/m²/год для домов >250 m²) |
| Особенность | Социальный компонент: уязвимые домохозяйства (EIKIS) |
| Статус Латвии | План на оценке Еврокомиссией (май 2026) |
| Источник | likumi.lv/ta/id/361681 |

**Что показывает калькулятор:**
- Предварительная пригодность: ELIGIBLE / LIKELY / UNLIKELY / NOT_ELIGIBLE / UNKNOWN
- Confidence: low / medium / high
- Объяснение, что нужно для пригодности (F-класс, доля EIKIS)
- Дисклеймер: «правила MK ещё не приняты, оценка приблизительная»

### 2. ALTUM remonta aizdevums

| Параметр | Значение |
|----------|----------|
| Тип | Кредит (без капитальной скидки) |
| Объём | от €10 000 |
| Ставка | 3.9% годовых |
| Срок | до 20 лет |
| Дедлайн на решение | до 30.06.2031 |
| Окно | `Atvērts` / Открыто |
| Заявитель | apsaimniekotājs / biedrība / ESCO |
| Назначение | ремонт, благоустройство, инженерные системы, проектная документация, проектное управление, надзор |
| Источник | altum.lv/pakalpojumi/iedzivotajiem/daudzdzivoklu-maju-remonta-aizdevums/ |

**Что показывает калькулятор:**
- Расчёт ежемесячного платежа на квартиру (по доле собственности)
- Сравнение с текущими счетами
- Чек-лист: что нужно для подачи (решение собственников о кредите, выбор уполномоченного лица, проектная документация)

**Это главный действующий путь сейчас, пока SCF не открылся.**

### 3. Коммерческий банк

| Параметр | Значение |
|----------|----------|
| Тип | Кредит (без субсидии) |
| Ставка | По текущему рынку (4–6% оценочно, без рекомендации конкретного банка) |
| Срок | до 25 лет |
| Окно | `Atvērts` (всегда доступно) |
| Заявитель | biedrība или собственники солидарно |

**Что показывает калькулятор:**
- Расчёт ежемесячного платежа
- Сравнение с ALTUM remonta aizdevums (обычно дороже)
- Когда имеет смысл: если ALTUM remonta aizdevums не подходит по объёму или дому нужны не только ремонтные работы

### 4. Свой ремонтный фонд (Pašu uzkrājums)

| Параметр | Значение |
|----------|----------|
| Тип | Накопления дома |
| Особенность | Без долговой нагрузки, но ограниченный объём |
| Окно | Всегда |

**Что показывает калькулятор:**
- Текущий баланс ремонтного фонда (если введён правлением)
- Что можно сделать на этот бюджет: окна, кровля, подъезды, частичные работы
- Прогноз накопления при разных взносах

### 5. Смешанный сценарий (Jaukts scenārijs)

Самый частый реалистичный путь: комбинация ALTUM remonta aizdevums + собственные средства + (когда откроется) грант SCF на компонент энергоэффективности.

**Что показывает калькулятор:**
- Распределение стоимости проекта по источникам
- Поэтапный план: что делается за свои, что за кредит, что за грант (когда откроется)

## Логика расчёта

Расчёт каждого сценария — детерминированная TypeScript-функция в `src/lib/financing/`.

**Входные данные:**
- Площадь дома (VZD)
- Энергокласс (BVKB)
- Серия здания
- Текущее потребление (из загруженных счетов или из BVKB)
- Размер ремонтного фонда (если введён правлением)
- Предполагаемый объём проекта (выбирается пользователем: «только окна», «утепление фасада», «полная реновация»)

**Расчёт стоимости:**
- На основе €/м² × площадь, с использованием норм ALTUM (€109–294/м² для проектов 2024)
- Корректировка по типу работ

**Расчёт экономии:**
- Физическая модель: коэффициенты по энергоклассам (G→C: ~65%, F→C: ~58%)
- Тариф SPRK ~0.085 €/kWh (синхронизируется)
- Расчётная экономия в €/мес. на квартиру

**Результат:**
- Стоимость проекта
- Доля гранта (если применимо)
- Доля собственников
- Распределение по квартирам (по доле собственности)
- Срок окупаемости
- Влияние на стоимость недвижимости (~10–11% по Latvijas Banka DP 3/2025)

## Что меняется относительно v1

| v1 | v2 |
|----|-----|
| Один «ALTUM subsidy calculator» | Пять сценариев с явным статусом окна |
| «Субсидия ALTUM покрывает 45%» | «Provizoriskais novērtējums: пригодность LIKELY/UNLIKELY с confidence» |
| «Срок окупаемости 4.2 года» | «Расчётный срок окупаемости 4.2 года при условиях X (ставка, объём проекта, тариф)» |
| «Начать процесс реновации» → голосование | «Подготовить дом» → переход к Decision Campaigns |
| Калькулятор ведёт к маркетплейсу + комиссия | Калькулятор ведёт к Readiness Report или Project Preparation Package |

---

## Зависимости

- **Данные:** VZD, BVKB, SPRK, загруженные пользователем счета
- **Питает:** Building Readiness Score (компонент `funding_eligibility_score` и `financial_feasibility_score`)
- **Использует:** rules engine — версионируемая логика правил по каждому сценарию

---

## Открытые вопросы

1. Точные правила пригодности SCF — выйдут в Q4 2026 (MK noteikumi). До этого — провизорные правила на основе текста плана.
2. Финансовые правила ALTUM remonta aizdevums и банков — версионируются в `src/lib/financing/rules/`, обновляются вручную.
3. Влияние ETS2 на расчёт экономии — добавить во вторую версию калькулятора, когда будут реальные цены CO₂.
