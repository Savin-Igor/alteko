## Открытые вопросы

### Решённые
- ~~**Юридическая сила Smart-ID голосования**~~ — **Действительно.** Dzīvokļa īpašuma likums, ст. 20(7-8), поправки 2022 г. Altum принимает электронные протоколы опроса. См. key-facts.md.
- ~~**Цифра ~23 500 зданий**~~ — **Верифицировано.** Это здания, *нуждающиеся в реновации*, а не весь фонд. Всего многоквартирных домов ~38 600 (CSB). Источник: Upitis et al. 2020, ALTUM fi-compass 2024.
- ~~**Конверсия Altum (<5%)**~~ — **Верифицировано: 2,7%.** 624 завершённых здания из 23 500. Источник: замглавы ALTUM, fi-compass, дек. 2024.
- ~~**+10–11% к стоимости квартиры**~~ — **Верифицировано.** Рабочий доклад Latvijas Banka DP 3/2025.
- ~~**Frontend-фреймворк**~~ — **Решено: Next.js + TypeScript, SSR.**

### Нерешённые
1. **Данные о собственниках из Zemesgrāmata** — публичного API нет. Фаза 3 (голосование) требует ручной загрузки CSV правлением или нотариальной выписки. См. integrations.md.
2. **Диапазон комиссии** — €2–10k (консервативно) vs €5–15k (исходные заметки); уточнить в ходе коммерческих переговоров с первыми подрядчиками.
3. **Расхождение показаний счётчиков 10–25%** — пока не верифицировано; проверить в отчётах Latvijas Ūdens или SPRK.
4. **Средняя стоимость реновации €200–500k** — верифицировать по данным закрытых проектов ALTUM.
